# Définition de Concepts de bases

Définition clés:

1.  [CPU] : (Prosseur) : Traite les opérations de l'ordinnateur (calcule, exécution d'application en temps réél)

2.  [RAM] : (Mémoire vive) : Stocke les donné temporairement utilisé par le CPU

3.  [Disque_dur] : Stockes les donné "inerte" sur le long therme

4.  [OS] : (Système d'exploitiation) : Ensemble de programe qui gère les ressources de l'ordinateur

5.  [Port] : Point virtuel utilisé pour établir et gérer des connections réseaux

6. [Protocole] : ensemble de règle standardisé qui definisse de formatage, la transimission et le traitement des données


Différense en Windows et Linux :

1.  Propriété :
        Windows : OS sous licence -> payant à usage unique
        Linux : OS Libre accès -> gratuit 

2.  Reconnaissance et compatibilité :
        Windows : Largement répendue, compatible native avec une grande majorité d'appareil
        Linux : Moin répendue, compatiblité amélioré mais toujours des dificulté pour certain appareils

3.  Personnalisation : 
        Windows : Peu personnalilsable, interface rigide
        Linux : Très autement personnalisable jusqu'au code source et modules

4.  philosophie :
        Windows : Convivial, standardisé, facile d'utilisation
        Linux : Plus brut, maléable, personnalisable sous toute ses couture

5.  Structure :
        Windows : Plusieurs lecteurs de fichiers...
        Linux : Arborecence unique


Réseaux :

1.  [IP] : Adresse réseaux de chaque appareil

2.  [DNS] : Requette de traduction d'un nom de dommaine en adresse ip
     ex : 
     www.exemple.com -> 192.168.1.1

3.  [NAT] : Mécanisme traduisant les adresses ip dans les paquets réseaux permettant a plusieurs appareil de partagé la même adresse

4.  [Protocoles_Majeurs] : Ensemble de règles régissant l'envoie et la réception de donnée
