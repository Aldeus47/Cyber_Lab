# Commande Linux testé et leurs utilitées :

1.  ls : Affiche l'arboressence de la machine
 [arboressence] : hierarchie interne des dosssiés et fichiés
     ex: Desktop Documents Downloads Music Pictures Public snap Templates Videos

2.  cd : Change le répertoire d'action
     ex : ~/Desktop$

3.  pwd : Affiche le reprtoir d'action
     ex : /home/******/Desktop

4.  cat : Visualise le contenu d'un ficher texte

5.  grep : Recherche de élément précis dans un fichier

6.  chmod : Change le mode de permission d'un seul fichier

7.  sudo : Donne les permission admistrateur pour un temps donné
